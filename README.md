#  Digital Library Audit System

## Project Description
This project is a SQL-based system designed to manage a digital library. It tracks book borrowing, identifies overdue books, and analyzes borrowing trends.

## Features
- Book and student management
- Issue and return tracking
- Overdue detection (14-day rule)
- Popular category analysis
- Inactive account cleanup

## Technologies Used
- SQL (Oracle / MySQL)

## Queries Included
1. Table Creation (DDL)
2. Data Insertion
3. Overdue Books Report
4. Popularity Index (GROUP BY)
5. Data Cleanup Query
